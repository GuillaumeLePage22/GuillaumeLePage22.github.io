Analyse de la composition de l'Assemblée Nationale

Objectif

Création d'un dashboard interactif sur Power BI pour analyser la composition de l'Assemblée Nationale française : partis, votes, expérience et âge des députés.

Visualisations principales

  • Composition de l'Assemblée par partis politiques
  • Votes des députés
  • Expérience et âge des élus
  • Analyse par parti et par député

Méthodologie

1. Collecte des données issues de data.gouv.fr

2. Web scraping des photos depuis le site de l'Assemblée Nationale

3. Structuration et nettoyage des données sur PowerQuery

4. Création du dashboard Power BI interactif

Technique

  • Web scraping : Python + BeautifulSoup
  • Data processing : PowerQuery
  • Data viz : Power BI

Analyse du nombres de crimes et délits enregistrés en France (entre 2016 et 2025)

Objectif

Création d'un dashboard interactif sur Power BI pour analyser l'évolution de la délinquance en France métropolitaine (2016-2025) : tendances par type d'infraction, profils victimes/mis en cause, et disparités territoriales.

Visualisations principales

  • Évolution temporelle des crimes/délits par type (violences, vols, stupéfiants...)
  • Profils victimes et mis en cause en 2025
  • Analyse géographique par département et région

Méthodologie

1. Collecte des données SSMSI/INSEE (victimes, infractions, mis en cause, véhicules)

2. Nettoyage et structuration via PowerQuery

3. Mesures DAX avancées : rangs dynamiques, variations %, top N par année/plage

4. Dashboard interactif Power BI

Techniques

  • Data processing : PowerQuery
  • Data viz : Power BI

Analyse des casteurs de la chaîne Twitch OTP

Objectif

Création d'un dashboard interactif sur Looker Studio pour analyser l'activité des casteurs de la chaîne Twitch OTP, spécialisée dans les compétitions League of Legends.

Visualisations principales

  • Équipes commentées par chaque casteur
  • Pauses/interruptions pendant les matchs
  • Duos de casteurs les plus fréquents
  • Position des casteurs (main desk, analyste...)
  • Nombre total de matchs commentés
  • Jours de diffusion de chaque casteur

Méthodologie

1. Web scraping sur gol.gg pour les données matchs

2. Analyse manuelle des VODs OTP (casteurs, positions, pauses)

3. Structuration sur Excel pour intégration à Looker Studio

4. Création du dashboard Looker Studio

Technique

  • Web scraping : Python + BeautifulSoup
  • Data processing : Pandas + Excel
  • Data viz : LookerStudio
